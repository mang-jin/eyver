# Eyver.sh

![Eyver Demo Image](eyver_demo.png)

Eyver means **Eye Saver**

```sh
Usage:
./eyver.sh <interval in seconds>
Example:
./eyver.sh 1200        # pops up Eyver every 20 minutes
```

You can change the message by editing the `eyver.sh` file.

## Build (POSIX systems)

Dependencies:
- GTK 4

```sh
cd eyver
./build.sh
```
